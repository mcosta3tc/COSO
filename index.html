Description de
l'Open Source

Un projet open source est un programme informatique dont le code source est à la disposition du grand public.
Il est développé par une communauté de programmeurs informatiques qui améliore le logiciel, corrige les bugs et les failles de sécurité.

ses
avantages

Prix: Les developpeurs mettent à disposition le code source de leur projet gratuitement.

Communauté: Les entreprises peuvent prendre leur indépendance par rapport à leurs éditeurs. Adapter le logiciel au plus-près des besoins de l'entreprise

Sécurité: Analysés par des d’experts du monde entier qui sont très réactifs. La moindre faille est rapidement détectée et corrigée. Il arrive que la communauté Open Source soit plus rapide qu’un fournisseur commercial classique pour répondre à une menace.
